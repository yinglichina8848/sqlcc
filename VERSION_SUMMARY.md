# SQLCC v0.2.1 版本总结

## 版本信息

- **版本号**：v0.2.1
- **发布日期**：2025-11-05 15:30:00
- **Git标签**：v0.2.1 (主分支)，v0.2.1-docs (文档分支)

## 版本概述

SQLCC v0.2.1 是一个重要的里程碑版本，主要引入了文档分支管理系统，完善了项目文档体系，并提供了完整的使用指南。本版本在保持原有功能稳定的基础上，大幅提升了项目的可用性和可维护性。

## 主要特性

### 1. 文档分支管理系统

- 创建了专门的docs分支用于管理Doxygen生成的API文档
- 实现了代码与文档的分离管理，提高了项目组织性
- 建立了分支间同步机制，确保代码和文档的一致性

### 2. 完善的文档体系

- **API文档**：完整的Doxygen生成的API文档，包含所有类和方法的详细说明
- **设计文档**：详细的存储引擎设计文档，解释系统架构和实现原理
- **测试文档**：全面的单元测试文档，说明测试框架和测试用例
- **使用指南**：完整的Guide.md，提供项目使用的详细说明
- **分支说明**：BRANCHES.md，详细解释项目分支结构和使用方法

### 3. 便捷的工具和脚本

- **Makefile**：提供便捷的构建、测试和文档生成命令
- **文档生成脚本**：自动化的文档生成流程
- **文档索引**：docs/index.md，作为所有文档的导航入口

## 技术架构

### 核心组件

1. **存储引擎** (StorageEngine)
   - 页式存储管理
   - 缓冲池和LRU替换策略
   - 磁盘I/O管理

2. **页面管理** (Page)
   - 8KB定长页面
   - 页面状态跟踪
   - 数据读写接口

3. **缓冲池** (BufferPool)
   - LRU替换策略
   - 页面缓存管理
   - 并发访问控制

4. **磁盘管理** (DiskManager)
   - 文件I/O操作
   - 页面读写
   - 错误处理

5. **日志系统** (Logger)
   - 简单有效的日志记录
   - 多级别日志支持
   - 可配置输出

### 测试框架

- 基于Google Test的单元测试
- 7个全面的测试用例
- 覆盖正常流程和边界条件
- 所有测试通过，功能稳定可靠

## 分支结构

- **主分支 (master)**：包含源代码、测试和基础文档
- **文档分支 (docs)**：包含主分支所有内容以及完整的Doxygen API文档

## 使用指南

### 查看代码

```bash
git clone https://gitee.com/yinglichina/sqlcc.git
cd sqlcc
# 默认在master分支，包含源代码和基础文档
```

### 查看API文档

```bash
git checkout docs
# 在浏览器中打开 docs/doxygen/html/index.html
```

### 构建和测试

```bash
make          # 构建项目
make test     # 运行测试
make docs     # 生成文档
```
