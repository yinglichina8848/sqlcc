# SQLCC 代码注释示例模板

## 1. 文件头部注释

```cpp
/**
 * @file filename.cpp
 * @brief 模块功能简要描述
 * @details 模块功能详细说明，包括设计思路、实现要点和使用场景
 * @author 作者名
 * @date 创建日期
 * @version 版本号
 * 
 * @par 版权信息
 * 版权所有 (c) 2024 SQLCC 项目组
 * 
 * @par 相关文件
 * @see filename.h - 对应的头文件
 * @see related_file.cpp - 相关实现文件
 * 
 * @par 数据库原理知识点
 * - 知识点1：详细说明
 * - 知识点2：详细说明
 * 
 * @par 设计模式
 * - 设计模式1：详细说明
 * - 设计模式2：详细说明
 */
```

## 2. 类注释

```cpp
/**
 * @class ClassName
 * @brief 类的简要描述
 * @details 类的详细说明，包括设计意图、核心功能和使用方法
 * 
 * @par 设计思路
 * - 设计目标：实现什么功能
 * - 核心算法：使用了什么算法
 * - 数据结构：使用了什么数据结构
 * - 与其他类的关系：继承、组合、依赖等
 * 
 * @par 数据库原理知识点
 * - 知识点1：详细说明
 * - 知识点2：详细说明
 * 
 * @par 示例用法
 * @code
 * ClassName obj;
 * obj.Method(param);
 * @endcode
 */
class ClassName {
    // 类成员和方法
};
```

## 3. 函数注释

```cpp
/**
 * @brief 函数简要描述
 * @details 函数详细说明，包括功能、算法、参数含义和返回值
 * 
 * @param param1 参数1的名称和含义
 * @param param2 参数2的名称和含义
 * @return 返回值的类型和含义
 * 
 * @par 算法复杂度
 * - 时间复杂度：O(logN)
 * - 空间复杂度：O(1)
 * 
 * @par 数据库原理知识点
 * - 知识点1：详细说明
 * - 知识点2：详细说明
 * 
 * @par 注意事项
 * - 注意事项1
 * - 注意事项2
 * 
 * @par 示例
 * @code
 * int result = FunctionName(10, 20);
 * @endcode
 * 
 * @see RelatedFunction - 相关函数
 * @throw ExceptionType 可能抛出的异常类型和条件
 */
ReturnType FunctionName(ParamType param1, ParamType param2) {
    // 函数实现
}
```

## 6. 枚举类型注释

```cpp
/**
 * @enum EnumName
 * @brief 枚举类型的简要描述
 * @details 枚举类型的详细说明，包括每个枚举值的含义和使用场景
 */
enum class EnumName {
    /**
     * @brief 枚举值1的简要描述
     * @details 枚举值1的详细说明
     */
    VALUE1,
    
    /**
     * @brief 枚举值2的简要描述
     * @details 枚举值2的详细说明
     */
    VALUE2
};
```

## 7. 宏定义注释

```cpp
/**
 * @def MACRO_NAME

 * 
 * @param param 宏参数的名称和含义
 */
#define MACRO_NAME(param) (param * 2)
```
