B+树内部节点类 

实现了\+B+树内部节点的功能，用于存储索引键和子节点指针

\begin{DoxyParagraph}{设计思路}

\begin{DoxyItemize}
\item 继承自\+BPlus\+Tree\+Node基类
\item 存储索引键和子节点指针，其中子节点指针数量比键数量多1
\item 提供序列化、反序列化、插入、删除、查询等接口
\item 支持自平衡操作，如分裂、合并等
\end{DoxyItemize}
\end{DoxyParagraph}
\begin{DoxyParagraph}{数据库原理知识点}

\begin{DoxyItemize}

\item 自平衡操作：通过分裂维持树的平衡
\end{DoxyItemize}
