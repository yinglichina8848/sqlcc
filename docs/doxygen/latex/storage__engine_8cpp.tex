\begin{DoxyParagraph}{数据库原理知识点}

\begin{DoxyItemize}
\item 存储引擎：实现了数据库的存储层，负责数据的持久化存储
\item 磁盘\+I/\+O管理：通过磁盘管理器实现高效的磁盘读写操作
\item 缓冲池管理：通过缓冲池提高数据库性能，减少磁盘\+I/\+O次数
\item 页面管理：实现了页面的创建、获取、更新、删除等操作
\item 事务支持：支持事务的\+ACID特性，确保数据的一致性和持久性
\end{DoxyItemize}
\end{DoxyParagraph}
\begin{DoxyParagraph}{设计模式}

\begin{DoxyItemize}
\item 单例模式：存储引擎在系统中通常是单例存在
\item 组合模式：存储引擎组合了磁盘管理器和缓冲池管理器
\item 策略模式：支持不同的缓冲池替换策略
\item 观察者模式：支持配置变更的通知机制 
\end{DoxyItemize}
