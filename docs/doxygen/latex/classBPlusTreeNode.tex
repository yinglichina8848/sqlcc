B+树节点基类 

定义了\+B+树节点的基本结构和接口，是内部节点和叶子节点的共同基类

\begin{DoxyParagraph}{设计思路}

\begin{DoxyItemize}
\item 采用继承模式，派生出内部节点和叶子节点
\item 每个节点对应一个磁盘页，通过页\+ID进行管理
\item 提供统一的接口，如序列化、反序列化、插入、删除、查询等
\item 支持自平衡操作，如分裂、合并等
\end{DoxyItemize}
\end{DoxyParagraph}
\begin{DoxyParagraph}{数据库原理知识点}

\begin{DoxyItemize}
\item B+树索引：实现了\+B+树节点的核心功能
\item 磁盘\+I/\+O优化：通过缓冲池减少磁盘\+I/\+O次数
\item 页管理：每个节点对应一个磁盘页，高效管理内存和磁盘数据
\end{DoxyItemize}
