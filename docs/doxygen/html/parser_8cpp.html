数据库原理知识点
  • SQL语法分析：实现了SQL语句的语法分析功能
  • 抽象语法树(AST)：将SQL语句转换为抽象语法树，便于后续处理
  • 词法分析：与词法分析器配合，将SQL文本转换为词法单元序列
  • 语法规则：实现了SQL语法规则的解析
  • 错误处理：处理语法错误，提供错误恢复机制
设计模式
  • 组合模式：使用组合模式构建抽象语法树
  • 访问者模式：支持AST节点的访问者模式，便于遍历和处理AST
  • 单例模式：解析器在系统中通常是单例存在
