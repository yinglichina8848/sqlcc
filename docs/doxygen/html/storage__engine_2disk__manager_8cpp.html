数据库原理知识点
  • 磁盘I/O管理：实现了高效的磁盘读写操作
  • 页面管理：负责页面的分配、释放、读写等操作
  • 文件管理：管理数据库文件的打开、关闭、同步等操作
  • 性能优化：支持批量读取和预取操作，提高磁盘I/O效率
  • 事务持久性：确保事务的持久性特性，保证已提交的事务数据不会丢失
设计模式
  • 单例模式：磁盘管理器在存储引擎中通常是单例存在
  • 策略模式：支持不同的预取策略
  • 观察者模式：支持配置变更的通知机制
