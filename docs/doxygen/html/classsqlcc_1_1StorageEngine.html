Detailed Description

存储引擎类，封装磁盘管理和缓冲池功能

Why: 数据库系统需要一个统一的接口来管理数据的存储和访问，隐藏底层实现的复杂性 What: StorageEngine类作为存储子系统的顶层接口，封装了磁盘管理和缓冲池功能，提供页面的创建、读取、写入和删除操作 How: 通过组合DiskManager和BufferPool对象，为上层提供统一的数据访问接口

Example:

ConfigManager config_manager;
StorageEngine storage_engine(config_manager);
Page* page = storage_engine.NewPage();
// ... use page ...
storage_engine.DeletePage(page->GetPageId());
