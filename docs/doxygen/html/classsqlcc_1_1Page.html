Detailed Description

数据库页面类，表示一个8KB的定长页面

Why: 数据库系统需要将数据组织成固定大小的页面，以便在磁盘和内存之间高效传输和管理 What: Page类封装了一个固定大小的数据块，包含页面ID和实际数据，是数据库存储系统的基本单位 How: 使用字符数组存储页面数据，提供读写操作接口，通过页面ID唯一标识每个页面
