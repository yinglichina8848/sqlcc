数据库原理知识点
  • 事务管理：实现了事务的ACID特性
  • 并发控制：使用锁机制确保多线程环境下的数据一致性
  • 死锁检测：实现了基于拓扑排序的死锁检测算法
  • 锁协议：支持共享锁和排他锁
  • 事务隔离级别：支持不同的事务隔离级别
  • 保存点：支持事务保存点功能
设计模式
  • 单例模式：事务管理器在系统中通常是单例存在
  • 观察者模式：支持事务状态变化的通知机制
  • 策略模式：支持不同的死锁检测和处理策略
