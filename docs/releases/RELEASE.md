# SQLCC 自动化发布

本项目提供了自动化发布脚本，可以简化版本发布流程。

## 快速开始

要发布新版本，只需运行以下命令：

```bash
./scripts/release.sh v0.2.4 "修复缓冲池溢出问题"
```

这将自动完成以下操作：
1. 更新版本号
2. 创建Git标签
3. 生成Doxygen文档
4. 生成代码覆盖率测试文档
5. 更新文档分支
6. 推送所有更改到远程仓库

## 详细说明

更多详细信息请参考 [自动化发布流程文档](docs/release_process.md)。

## 注意事项

- 在运行脚本前，请确保所有代码更改已提交
- 脚本会自动处理分支切换和文档复制
- 如果需要回滚发布，请参考文档中的手动回滚部分
