# SQLCC

#### 介绍
开发语言 C / C++ 
AI-IDE 字节 Trae (多模态，无限时长) + Aider 
存储 自研页式文件管理，使用本地文件
模块 核心功能要求 自动化测试基准
存储引擎 8KB定长页，空间管理，定/变长记录
SQL解析 解析指定6种SQL语句（CRUD），生成正确AST 全部测试SQL解析通过，AST打印正确
索引 B+树，单字段唯一，支持=, >, <, 范围查询
